<Section icon={Zap} title="Langkah demi Langkah: Farming Batu Abyssal Umbral">

Untuk mengumpulkan **batu abyssal umbral clover retribution** dan material terkaitnya secara efisien, ikuti loop yang dioptimalkan ini. Proses ini meminimalkan waktu henti dan memaksimalkan peluang Anda untuk mendapatkan drop langka.

<Steps>
  <Step num="1" title="Persiapkan Build Anda">
    Lengkapi build Divine Restriction untuk keamanan atau build Dark Magic INT tinggi. Pastikan Anda memiliki Standard atau Copper Pickaxe di inventaris Anda, karena beberapa node Abyssal mungkin muncul di lingkungan raid.
  </Step>
  <Step num="2" title="Mulai Raid">
    Pergi ke pintu merah dan hitam di utara tengkorak raksasa. Jika farming solo, pilih Mode Mudah. Jika dalam grup terkoordinasi yang mencari batu secara langsung, pilih Mode Sulit untuk memicu bonus tabel loot 2026.
  </Step>
  <Step num="3" title="Lakukan Cutscene Jump">
    Segera setelah layar pemuatan berakhir, lompat dari platform spawn dan berjalan maju selama cutscene. Ini memberi Anda keunggulan 5 detik untuk mencapai tengah sebelum Medusa memulai penyelaman bawah tanah pertamanya.
  </Step>
  <Step num="4" title="Terapkan Reset Cheese">
    Ketika Medusa menyelam di bawah tanah, minta orang dengan ancaman tertinggi (biasanya pemain solo) untuk mereset karakternya. Ini memaksa bos untuk segera respawn di tengah, melewati fase parkour yang membosankan.
  </Step>
  <Step num="5" title="Kumpulkan dan Buat">
    Kumpulkan Gorgon Token dan Mystical Serpent Eyes. Bawa ini ke Orion atau Blacksmith level 74 untuk membuat bagian Abyssal terakhir Anda menggunakan Batu Abyssal Umbral sebagai agen pengikat terakhir.
  </Step>
</Steps>
