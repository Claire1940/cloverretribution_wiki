### Perpustakaan AFK untuk Spin Pasif

Perpustakaan AFK adalah zona khusus tempat Anda dapat mengumpulkan hadiah, termasuk spin, saat Anda tidak berada di depan keyboard.

| Fitur          | Deskripsi                                       | Tingkat Spin (sekitar)     | Catatan                                            |
| :------------- | :------------------------------------------------ | :------------------------- | :------------------------------------------------- |
| **Standar**    | Generasi spin pasif dasar.                        | 4 spin per jam             |                                                    |
| **Pengguna Premium** | Menggandakan tingkat spin standar.                | 8 spin per jam             | Membutuhkan Roblox Premium.                        |
| **VIP Game Pass** | Menggandakan tingkat spin standar.                | 8 spin per jam             | Dapat dibeli dalam game.                           |
| **Gabungan**   | Premium + VIP Game Pass.                          | 16 spin per jam            | Tingkat spin pasif maksimum.                       |

<Callout type="tip" title="Otomatiskan Farming AFK Anda">
Gunakan auto-clicker (seperti TinyTask atau aplikasi Windows Store) untuk mencegah Anda dikeluarkan karena tidak aktif. Ini memastikan generasi spin berkelanjutan saat Anda AFK.
</Callout>

### Sistem Prestise untuk Pembayaran Spin Besar

Sistem Prestise memungkinkan Anda untuk mengatur ulang level karakter Anda setelah Anda mencapai level 405. Ini menawarkan sejumlah besar spin berdasarkan level Anda saat ini.

#### Opsi dan Hadiah Prestise

| Opsi                                  | Efek                                                                                               | Catatan                                                                         |
| :------------------------------------ | :------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------ |
| **Grow Closer to Grimoire**           | Kesempatan untuk membuat grimoire Anda saat ini menjadi berdaun empat. Pity terbentuk di level maks (10% + 10% per reset). | Terutama untuk peningkatan Grimoire, bukan spin langsung.                       |
| **Another Chance at Life**            | Dapatkan lima poin stat permanen.                                                                  | Berfokus pada pertumbuhan stat, bukan spin.                                     |
| **Don't Want to Part Ways with Grimoire** | Memberikan Arcane Locket untuk menyimpan sihir Anda.                                               | Berguna untuk penyimpanan sihir, bukan spin.                                    |
| **If Only I Had a Few More Chances**  | **Memberikan spin yang setara dengan level Anda dibagi tiga.**                                     | **Ini adalah opsi untuk farming spin.**                                         |

<Callout type="warning" title="Persiapan Prestise">
Sebelum melakukan reset Prestise, pastikan semua peralatan yang Anda inginkan telah dilengkapi. Setelah reset, Anda akan berada di level satu dan mungkin tidak memenuhi persyaratan level minimum untuk melengkapi kembali item level tinggi, yang berpotensi menghambat siklus peningkatan level Anda berikutnya.
</Callout>
