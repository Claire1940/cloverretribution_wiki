To successfully copy a spell, you must target another player and use the imitation ability. However, you can only copy spells that you have the **Mastery** for. If a spell requires 150 Mastery and your Key Magic is only at level 50, the copy will fail. Furthermore, you do not inherit the passive abilities of the original magic, meaning an imitated Dark spell may naturally deal less damage than one cast by a dedicated Dark user.

| Mechanic | Description | Restriction |
| :--- | :--- | :--- |
| **Copy Limit** | Up to 4 spells with 4-Leaf | Cannot copy the same move twice |
| **Mastery Requirement** | Must match the spell's unlock level | Cannot copy what you haven't "earned" |
| **Passive Inheritance** | Only active spells are copied | No elemental passives (e.g., Burn, Shock) |
| **Targeting** | Must interact with a live player | Cannot copy from other Key/Imitation users |

<Section icon={Target} title="The Arcane Doors: Slot Modifiers">

The core of your build revolves around the **Arcane Doors**. Each slot in your grimoire applies a specific multiplier to the spell placed within it. Understanding these trade-offs is the difference between a functional build and one that runs out of mana in seconds.

| Slot Name | Unlock Condition | Mana Modifier | Cooldown Modifier |
| :--- | :--- | :--- | :--- |
| **Arcane Door 1** | Default | **-25% Cost** | 3x Increase |
| **Arcane Door 2** | Default | 3x Increase | **-25% Cooldown** |
| **Arcane Door 3** | 150 Mastery | 2x Increase | 2x Increase |
| **Arcane Door 4** | 4-Leaf + 150 Mastery | **-25% Cost** | **-25% Cooldown** |

<Callout type="warning" title="The Slot 3 Trap">
Arcane Door 3 is generally considered the weakest slot because it nerfs both mana consumption and cooldown. Use this slot for utility spells or heals that aren't critical to your main DPS rotation.
</Callout>

Selecting which spell goes into which door is a strategic puzzle. For example, a buff that lasts a long time but has a high mana cost is perfect for **Arcane Door 1**, as the tripled cooldown won't matter if the buff duration is sufficient. Conversely, your primary "spammable" DPS move should always be reserved for **Arcane Door 4**.

</Section>
