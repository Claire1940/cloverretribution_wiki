<Section icon={Target} title="Como Obter e Rerrolar os Melhores Traços">

Obter o **melhor traço de Clover Retribution** requer uma combinação de sorte e gerenciamento de recursos. Os jogadores usam "Giros" para rerrolar seus traços. Como os traços Míticos têm uma taxa de queda inferior a 1%, entender o processo de rerrolagem é vital para uma progressão eficiente.

<Steps>
  <Step num="1" title="Acumule Giros">
    Colete giros através de missões diárias, eventos mundiais ou usando códigos promocionais ativos. Guarde-os até ter pelo menos 50-100 giros para uma melhor chance de traços de alto nível.
  </Step>
  <Step num="2" title="Identifique o Caminho da Build">
    Decida se você vai para uma build Mágica, Corpo a Corpo ou Híbrida. Isso evita que você pule um ótimo traço como **Heavenly Restricted** se você pretendia ser um guerreiro.
  </Step>
  <Step num="3" title="Raridade Alvo">
    Se você rolar um traço Lendário como **Boss Slayer** ou **Loved by Mana**, considere mantê-lo a menos que você tenha um excedente massivo de giros. Estes são frequentemente "bons o suficiente" para todo o conteúdo do fim de jogo.
  </Step>
  <Step num="4" title="Finalize com o Equipamento">
    Depois de ter o traço desejado, equipe armaduras que complementem suas estatísticas. Por exemplo, use equipamento de regeneração de Mana com **Monstrous Reserves** para se tornar uma máquina de conjuração.
  </Step>
</Steps>

<Callout type="info" title="Dica Profissional">
O traço **Prodigy** é o melhor para nivelamento. Se você o rolar cedo, mantenha-o até atingir o limite de nível, então rerrole para um traço Mítico ou Lendário focado em combate.
</Callout>
